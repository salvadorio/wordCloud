Social Media and Digital Marketing class had an assignment... better understand how others see you, compared to how you see yourself, using the Johari Window model? 

We're supposed to ask ~20 people for 10 words they'd use to describe us, and then put them into a word cloud. 

But, I feel a word cloud woudln't organize the data very well - I don't *only* care about the number of times someone described me as something. 
I care about showing who describes me as what, and showing overlaps between people's opinions. And looking for more general patterns?

So my vibe-coded word map generator website lets you add people and the list of words they use to describe you, and then uses a spring-weighted node net to organize them.

You can organize words by who described you as them, or by their semantic similarity.
